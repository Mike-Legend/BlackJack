Lab 4: Blackjack Game Submitting


How to Submit Your Lab

  1. Get your code into GitHub

    To be graded, your code must be in GitHub. On the Git Changes tab in Visual Studio...
    1. enter a message in the text box above the Commit All button
    2. click the Commit All button
    3. click the up arrow (push) button

    More details about Committing and Pushing your code to GitHub

  2. Download the zip from GitHub

    On the Code page of your repo in GitHub...
    1. Click the green Code button
    2. Click the Download ZIP button

  3. Submit the zip file

    Take the zip file that you downloaded from GitHub and upload that to the FSO assignment.
